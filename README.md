# 📄 Hoja de Vida - HTML y CSS

## 📌 Descripción

Este proyecto consiste en el desarrollo de una hoja de vida (Currículum Vitae) utilizando únicamente **HTML5** y **CSS**. El objetivo es poner en práctica conceptos fundamentales de maquetación web, organización de contenido y diseño responsivo.

## 🚀 Tecnologías utilizadas

- HTML5
- CSS
- Visual Studio Code

## 📁 Estructura del proyecto

```
Proyecto/
│── index.html
│── styles.css
│── img/
│   └── perfil.jpg
└── README.md
```

## 🎯 Características

- Foto de perfil circular.
- Información personal.
- Perfil profesional.
- Estudios básicos.
- Estudios profesionales.
- Idiomas.
- Habilidades técnicas (Skills).
- Diseño limpio y organizado.

## 📷 Vista previa

La hoja de vida contiene las siguientes secciones:

- Información personal
- Perfil profesional
- Formación académica
- Idiomas
- Habilidades

## ⚙️ Instalación

1. Clona el repositorio.

```bash
git clone https://github.com/usuario/repositorio.git
```

2. Ingresa al proyecto.

```bash
cd repositorio
```

3. Abre el archivo `index.html` en tu navegador.

## 💻 Ejecución

No requiere instalación de dependencias.

Solo abre el archivo:

```
index.html
```

## 📚 Conceptos aplicados

- Etiquetas semánticas HTML.
- Selectores CSS.
- Flexbox.
- Bordes y bordes redondeados.
- Espaciado (Margin y Padding).
- Organización mediante contenedores.
- Diseño adaptable.
